# 🛋️ Elammary CRM — Furniture Management System

**Elammary CRM** is a modern, serverless Customer Relationship Management (CRM) platform designed specifically for furniture businesses. Built with **React** and powered by **Firebase**, it allows you to manage customers, track orders, and monitor your business analytics in real-time.

## 🚀 Features
- **Dashboard Analytics**: Real-time sales stats, revenue tracking, and customer distribution charts.
- **Customer Management**: Detailed profiles, tiered loyalty system, and purchase history.
- **Order Tracking**: Manage orders from pending to delivered with automatic tier updates.
- **Product Catalog**: Manage your furniture inventory with ease.
- **Serverless Architecture**: Fully scalable and ready for Vercel deployment.

## 🛠️ Tech Stack
- **Frontend**: React 18, Tailwind CSS (Lucide Icons)
- **Backend/Database**: Firebase Firestore
- **Authentication**: Firebase Auth
- **Hosting**: Optimized for Vercel

## ⚙️ Installation & Setup

1. **Clone the Repo**
   ```bash
   git clone https://github.com/Seif-83/Elammary.git
   cd Elammary
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Firebase Configuration**
   The app is pre-configured with the Firebase SDK. Ensure your Firestore rules allow read/write access for authenticated users.

4. **Run Locally**
   ```bash
   npm start
   ```

5. **Deployment**
   Connect this repository to your Vercel account for automatic deployments.
